# GenAI FAQs & Ethical Use Website

## Project title
GenAI FAQs & Ethical Use — Static Website

## Short overview
A small, responsive static website explaining Generative AI, its FAQs, ethical use guidelines (with emphasis on academe), and a real-world analogy comparing AI-assisted development to hiring a wedding documentation team.

## How to run
1. Download and unzip the project.
2. Open `index.html` in your web browser (no server required).

## Frameworks / libraries used
- Bootstrap 5 (CDN)
- Bootstrap Icons (CDN)
- Plain HTML, CSS, JavaScript

## Folder structure
```
/genai-website
  /assets
    /css/styles.css
    /js/script.js
  index.html
  faq.html
  ethics.html
  article.html
  README.md
```
